## 🚧 Lo que estoy construyendo ahora

### 1) Sistemas de automatización para creadores

- **AutoPubMonitor / LazyEdit / AutoPublication**  
  Cadena de herramientas conectada para convertir ideas en bruto en contenido publicado:  
  transcription -> subtitles -> highlights -> metadata -> multi-platform distribution
- **AutoAppDev / AutoNovelWriter**  
  Automatización estructurada para scaffolding de apps y escritura de formato largo

Enlaces de proyectos:
- https://github.com/lachlanchen/AutoPubMonitor
- https://github.com/lachlanchen/LazyEdit
- https://github.com/lachlanchen/AutoPublication
- https://github.com/lachlanchen/AutoAppDev
- https://github.com/lachlanchen/AutoNovelWriter

### 2) Sistemas de idioma y aprendizaje

- **EchoMind**: voz + chat multilingüe para pensar y aprender
- **MultilingualWhisper**: detección de idioma a nivel de segmento para medios multilingües  
  https://github.com/lachlanchen/MultilingualWhisper
- **WordOrigins**: etimología como un grafo vivo del lenguaje

### 3) Imagen, óptica e IA corporizada

- **OpenHI**: imagen hiperespectral neuromórfica
- **LightMind**: sistema wearable de memoria con enfoque privacy-first
- **IdeasGlass**: gafas de IA para capture -> translate -> publish
- **AgInTi**: exploración de agente de IA humanoide  
  https://github.com/lachlanchen/AgInTi

Esta capa conecta percepción física, computación y cognición personal.
