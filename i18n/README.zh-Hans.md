# Lazying 的艺术 —— 用更少的工作创造价值

你好，我是 <b>Lachlan</b>。

我在构建工具、社区与一套实用哲学，帮助人们<b>减少无谓的忙碌</b>，把时间让给更有意义、更投入的生活。  
<b>北极星：</b>减少不必要的劳动，让每个人都能不被工作困住也能生存并成长。

> <b>Only Ideas.</b> 做最少但最解锁生活的部分。

---

## 为什么存在

- 多数“效率”只是在增加待办。<b>Lazying</b> 是减法：自动化、简化、守护注意力。
- 工具应<b>减少摩擦</b>，而不是增加负担。
- 默认<b>本地优先、隐私优先</b>。

## 重点项目（在线 & 打磨中）

- <b>EchoMind</b> —— 多语音/文本学习与创作。  
  <a href="https://chat.lazying.art">chat.lazying.art</a> <a href="https://chat.lazying.art"><img src="../figs/go_button.svg" alt="Go" height="18" style="vertical-align: middle;"/></a> · [文档](../projects/EchoMind.md)

- <b>MultilingualWhisper</b> —— 基于 Whisper 的多语分段检测增强版。  
  [文档](../projects/MultilingualWhisper.md)

- <b>LazyEdit</b> —— “一键”AI 视频编辑：转录、字幕、高光、词卡、元数据、封面。  
  [文档](../projects/LazyEdit.md)

- <b>VideoCaptioner</b>  
  · <b>ViT‑GPT2 版</b> —— 关键帧提取 + 字幕；CLI + Python 包。  
  · <b>CLIP+GPT 版</b> —— CLIP 视觉表征 + GPT 风格解码生成图像/视频描述。  
  [文档](../projects/VideoCaptionerWithVit.md) · [文档](../projects/VideoCaptionerWithClip.md)

- <b>AutoPublication</b> —— 从草稿到发布的端到端流水线（含 <b>AutoPubMonitor</b> 队列/缓存/多平台同步）。  
  [文档](../projects/AutoPublication.md) ·（[监控文档](../projects/AutoPubMonitor.md)）

- <b>WordOrigins</b> —— 以交互图探索多语言词源。  
  [文档](../projects/WordsOrigin.md)

- <b>IdeasGlass</b> —— AI 眼镜：捕捉、翻译并自动产出可发布的高光内容。  
  <a href="https://github.com/lachlanchen/IdeasGlass">github.com/lachlanchen/IdeasGlass</a> · [文档](../projects/IdeasGlass.md)

- <b>LazyEarn</b> —— 把小贡献转化为收益的自动化。  
  <a href="https://github.com/lachlanchen/LazyEarn">github.com/lachlanchen/LazyEarn</a>
