# 게으름의 기술 — 적은 일로 더 큰 가치

안녕하세요, <b>라클란</b>입니다.

저는 도구와 커뮤니티, 실용 철학을 통해 사람들이 <b>불필요한 바쁨을 줄이고</b> 더 의미 있는 삶을 살도록 돕습니다.  
<b>북극성:</b> 쓸데없는 노동을 줄여, 일에 갇히지 않고도 살아갈 수 있게.

> <b>Only Ideas.</b> 최소로 만들어 최대를 열어라.

---

## 왜 존재하나요

- 대부분의 “생산성”은 할 일을 더 늘립니다. <b>Lazying</b>은 감산: 자동화, 단순화, 주의 보호.
- 도구는 <b>마찰을 줄여야</b> 합니다.
- <b>로컬/프라이버시 우선</b>이 기본입니다.

---

## 대표 프로젝트 (운영/개발 중)

- <b>EchoMind</b> — 다국어 음성/채팅 학습과 창작.  
  <a href="https://chat.lazying.art">chat.lazying.art</a> <a href="https://chat.lazying.art"><img src="../figs/go_button.svg" alt="Go" height="18" style="vertical-align: middle;"/></a> · [docs](../projects/EchoMind.md)

- <b>MultilingualWhisper</b> — Whisper 다국어 구간 감지 강화.  
  [docs](../projects/MultilingualWhisper.md)

- <b>LazyEdit</b> — “원클릭” AI 영상 편집: 자막/하이라이트/워드카드/메타데이터/표지.  
  [docs](../projects/LazyEdit.md)

- <b>VideoCaptioner</b>  
  · <b>ViT‑GPT2</b> — 키프레임 + 캡션, CLI + Python.  
  · <b>CLIP+GPT</b> — CLIP 시각 임베딩 + GPT식 디코더로 캡션.  
  [docs](../projects/VideoCaptionerWithVit.md) · [docs](../projects/VideoCaptionerWithClip.md)

- <b>AutoPublication</b> — 초안→게시 파이프라인 (<b>AutoPubMonitor</b> 큐/캐시/멀티 플랫폼).  
  [docs](../projects/AutoPublication.md) · ([monitor](../projects/AutoPubMonitor.md))

- <b>WordOrigins</b> — 다국어 어원 인터랙티브 그래프.  
  [docs](../projects/WordsOrigin.md)

- <b>IdeasGlass</b> — 캡처·번역·채널용 하이라이트 자동 생성 AI 안경.  
  <a href="https://github.com/lachlanchen/IdeasGlass">github.com/lachlanchen/IdeasGlass</a> · [docs](../projects/IdeasGlass.md)

- <b>LazyEarn</b> — 작은 기여를 수익으로 바꾸는 자동화.  
  <a href="https://github.com/lachlanchen/LazyEarn">github.com/lachlanchen/LazyEarn</a>

- <b>IdeasRobot</b> — 아이디어를 초안·작업·예약 게시로 바꾸는 에이전트.  
  <a href="https://github.com/lachlanchen/IdeasRobot">github.com/lachlanchen/IdeasRobot</a>
