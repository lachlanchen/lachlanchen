## ▶️ 사용법

기본 워크플로:

1. `README.md`를 수정합니다(기준 영어/루트 프로필 콘텐츠).
2. `i18n/`의 언어 파일을 동기화합니다.
3. 링크와 섹션을 확인합니다.
4. README 전용 업데이트를 배포할 때는 `README.md`만 커밋합니다.

README 전용 푸시 헬퍼:

```bash
scripts/push_readme_only.sh -m "Update profile README" -b main
```

스크립트 동작:

1. `origin`을 가져옵니다.
2. `origin/<branch>`에서 임시 worktree를 생성합니다.
3. 루트 `README.md`를 임시 worktree로 복사합니다.
4. `README.md`만 커밋하고 푸시합니다.
5. 임시 상태를 정리합니다.

## ⚙️ 구성

- 언어 옵션은 각 README 변형 파일 상단에 **단일 언어 선택 줄**로 유지합니다.
- 현재 루트 README 링크 대상:
  - `i18n/README.ar.md`, `README.es.md`, `README.fr.md`, `README.ja.md`, `README.ko.md`, `README.vi.md`, `README.zh-Hans.md`, `README.zh-Hant.md`, `README.de.md`, `README.ru.md`
- 스폰서십 링크는 `.github/FUNDING.yml`에 설정되어 있습니다.
