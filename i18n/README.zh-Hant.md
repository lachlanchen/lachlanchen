# Lazying 的藝術 —— 以更少工作創造價值

嗨，我是 <b>Lachlan</b>。

我在打造工具、社群與一套實用的哲學，幫助大家<b>減少無謂忙碌</b>，把時間留給更有意義、更投入的生活。  
<b>北極星：</b>減少不必要的勞動，讓每個人都能不被工作困住也能生存並發展。

> <b>Only Ideas.</b> 做最少、但最能解鎖生活價值的部分。

---

## 為什麼存在

- 多數「生產力」只是在加待辦。<b>Lazying</b> 是減法：自動化、簡化、守護注意力。
- 工具應<b>移除摩擦</b>，而非增加負擔。
- 預設<b>本地優先、隱私優先</b>。
