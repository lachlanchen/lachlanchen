## ▶️ الاستخدام

سير العمل الأساسي:

1. عدّل `README.md` (المحتوى الأساسي بالإنجليزية/الجذر للملف الشخصي).
2. حافظ على توافق ملفات اللغات داخل `i18n/`.
3. تحقّق من الروابط والأقسام.
4. عند النشر، نفّذ commit لملف `README.md` فقط في تحديثات README-only.

مساعد دفع README فقط:

```bash
scripts/push_readme_only.sh -m "Update profile README" -b main
```

سلوك السكربت:

1. يجلب من `origin`.
2. ينشئ worktree مؤقتًا من `origin/<branch>`.
3. ينسخ ملف `README.md` من الجذر إلى الـ worktree المؤقت.
4. يجري commit ثم push لملف `README.md` فقط.
5. ينظّف الحالة المؤقتة.

## ⚙️ الإعدادات

- خيارات اللغات تُحفظ كسطر **واحد لاختيار اللغة أعلى** كل نسخة README.

  - `README.md` (الإنجليزية/الجذر)
