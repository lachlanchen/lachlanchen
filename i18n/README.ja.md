# Lazying の作法 — 少ない労力で価値を生む

こんにちは、<b>Lachlan</b> です。

私はツールとコミュニティ、そして実践的な哲学を作り、<b>無駄な作業を減らし</b>、より意味のある生き方を支えることに取り組んでいます。  
<b>北極星：</b> 不要な労働を減らし、仕事に縛られずに生きられるように。

> <b>Only Ideas.</b> 最小で、人生を最大に解放する。

---

## なぜ存在するか

- 多くの「生産性」はタスクを増やすだけ。<b>Lazying</b> は減算：自動化・単純化・注意の保護。
- ツールは<b>摩擦を減らす</b>べきで、増やすべきではない。
- デフォルトは<b>ローカル優先・プライバシー優先</b>。

---

## 主なプロジェクト（公開・開発中）

- <b>EchoMind</b> — 多言語の音声/チャット学習と創作。  
  <a href="https://chat.lazying.art">chat.lazying.art</a> <a href="https://chat.lazying.art"><img src="../figs/go_button.svg" alt="Go" height="18" style="vertical-align: middle;"/></a> · [docs](../projects/EchoMind.md)

- <b>MultilingualWhisper</b> — Whisper の多言語セグメント検出を強化。  
  [docs](../projects/MultilingualWhisper.md)

- <b>LazyEdit</b> — 「ワンクリック」AI 動画編集：書き起こし、字幕、ハイライト、単語カード、メタデータ、カバー。  
  [docs](../projects/LazyEdit.md)

- <b>VideoCaptioner</b>  
  · <b>ViT‑GPT2 版</b>：キーフレーム抽出 + 字幕、CLI + Python。  
  · <b>CLIP+GPT 版</b>：CLIP 視覚表現 + GPT デコーダで画像/動画キャプション。  
  [docs](../projects/VideoCaptionerWithVit.md) · [docs](../projects/VideoCaptionerWithClip.md)

- <b>AutoPublication</b> — 草稿から公開まで（<b>AutoPubMonitor</b> のキュー・キャッシュ・多プラットフォーム同期）。  
  [docs](../projects/AutoPublication.md) ·（[monitor](../projects/AutoPubMonitor.md)）

- <b>WordOrigins</b> — 多言語の語源をインタラクティブに。  
  [docs](../projects/WordsOrigin.md)

- <b>IdeasGlass</b> — アイデアを捉え、翻訳し、チャンネル向けのハイライトを自動生成する AI メガネ。  
  <a href="https://github.com/lachlanchen/IdeasGlass">github.com/lachlanchen/IdeasGlass</a> · [docs](../projects/IdeasGlass.md)
