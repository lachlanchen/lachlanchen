## 🚀 Usage

### Update this profile README directly

- Edit sections in `README.md` directly.
- Keep substantive content while avoiding duplicate blocks (especially banner/support panel).

### Use the provided scripts

```bash
bash scripts/update-read-me/run_lachlanchen_only.sh
```

After validating updates, publish only the README:

```bash
bash scripts/push_readme_only.sh
```

## 🧪 Troubleshooting

- Duplicate banner/support blocks appear after pipeline runs: remove duplicates and keep only one canonical block each.
- Broken quick-link translations: verify target files exist in `i18n/`.
- Script failures usually come from missing `bash` or `rg` in PATH; install these dependencies and rerun from repository root.
- If a link points to a stale branch or repository name, update it to the canonical repo path.

## 🧰 Development Notes
